  • P1.1 — เขียน Design Intent Statement

    เขียน 3–5 ประโยค อธิบายว่าบ้านนี้ให้ความรู้สึกอะไร ใครอยู่ วิถีชีวิตเป็นยังไง เช่น "บ้านที่เงียบสงบและหนักแน่น เปิดรับแสงเช้า สวนเป็นส่วนหนึ่งของการอยู่อาศัย"

    ✍️ วาดมือ 📝 Journal
    W1–W2

  • P1.2 — Precedent Study (5 โปรเจกต์)

    เลือกสถาปนิก 5 คน วิเคราะห์แต่ละโปรเจกต์ด้วย 4 หัวข้อ: Form, Light, Material, Circulation แล้ว "วาดลอก" Floor Plan มือ — วิธีที่ Tadao Ando ใช้จริง

    🖊️ Hand Trace 📐 Scale Drawing 🗂️ Miro / IdeaFlow
    W1–W3

  • P1.3 — Bubble Diagram + Zoning

    วาด Bubble ให้ทุกห้อง เชื่อมเส้นความสัมพันธ์ จากนั้น Cluster เป็น Zone: Public (ห้องนั่งเล่น, ครัว), Private (ห้องนอน), Service (ห้องน้ำ, Storage). ทำบนกระดาษ A3 อย่างน้อย 2 เวอร์ชัน

    ✏️ กระดาษ A3 🔵 Bubble Method
    W2–W3

  • P1.4 — Sun Path + Wind Rose Analysis

    ใช้ App "Sun Seeker" หรือ SunCalc.org เช็ค Sun Path ของพื้นที่ที่จะสร้าง วาด Diagram แสดงทิศที่แดดส่อง และทิศลมหลักของไทย (ลมมรสุม NE/SW) เพื่อกำหนดการวางบ้านบนที่ดิน

    ☀️ Sun Seeker App 🌬️ Wind Rose 🗺️ SunCalc.org
    W3–W4

  • P1.5 — Concept Board + Concept Statement Lock

    รวม Reference, Mood, Materials, Colors บน Mood Board เดียว จากนั้นเขียน Concept Statement 1 ประโยคที่จะเป็น "เข็มทิศ" ตลอดโปรเจกต์ Print หรือวางบนโต๊ะให้เห็นตลอด

    🖼️ Canva / Miro 🖨️ Print & Pin

Design Methods
วิธีการออกแบบแบบสถาปนิกชั้นนำ

Trace Paper Method

TADAO ANDO · ZAHA HADID
  • วาง Trace Paper ทับแบบเดิม
  • วาดใหม่ทับ เก็บส่วนที่ดี แก้ส่วนที่ไม่ดี
  • ซ้อนได้หลายชั้น เห็น Evolution ของแบบ
  • ห้ามลบแบบเก่า — เก็บทุก Layer
"I draw everything by hand. The hand and the mind must work together." — Tadao Ando

3 Options Rule

STANDARD PRACTICE · ALL FIRMS
  • วาด Option A (แบบที่นึกได้แรก)
  • วาด Option B (ตรงข้ามกับ A ที่สุด)
  • วาด Option C (Hybrid หรือ Radical)
  • เปรียบเทียบด้วย Criteria Matrix แล้วเลือก
ป้องกันการติด "First Idea Bias" — แบบแรกไม่ใช่แบบที่ดีที่สุดเสมอไป

Programmatic Zoning

SPACE PLANNING METHOD
  • แบ่งทุกห้องเป็น 3 Zone: Public / Private / Service
  • Public (นั่งเล่น, ครัว) → ติดถนน/สวนหน้า
  • Private (นอน, Study) → ส่วนลึก เงียบ
  • Service (น้ำ, Storage) → ตรงกลาง หรือด้านหลัง
เหมือนกับ Lean Layout ใน Industrial Engineering — minimize movement distance

Section as Generator

KENGO KUMA · BJARKE INGELS
  • แทนที่จะเริ่มจากผัง เริ่มจากวาด Section ก่อน
  • Section แสดงความสูง, แสง, และ Volume ได้ดีกว่า
  • วาด Section ผ่านห้องที่สำคัญที่สุดก่อน
  • ผังตามมาภายหลังจาก Section Logic
สำหรับบ้าน Tadao Ando style — Section คือที่ที่แสงถูกออกแบบ

Precedent Trace Analysis

ARCHITECTURAL EDUCATION METHOD
  • พิมพ์ Floor Plan ของสถาปนิกที่ชอบ
  • วาง Trace Paper ทับ วาดลอก Scale จริง
  • เขียน Annotation: ทำไมห้องนี้อยู่ตรงนี้?
  • ดูว่ามี Pattern อะไรที่ดึงมาใช้กับบ้านตัวเองได้
สถาปนิก IDIN Architects ไทยแนะนำวิธีนี้ในทุก Workshop

Design Criteria Matrix

RATIONAL DECISION METHOD
  • ตั้ง Criteria: แสง, Circulation, Concept, Cost
  • ให้ Weight แต่ละ Criteria (1–5)
  • Score แต่ละ Option ในแต่ละ Criteria
  • Option ที่ Score สูงสุด = เลือก (แต่ใช้ Gut ด้วย)
ประโยชน์ของ IE Background — ตัดสินใจอย่างมี Logic ไม่ใช่แค่ความรู้สึก
